Vent2U Index

Project description:
In this project I worked with Nikolaj, Mathias, Anina and partially Linh
In the Vent 2 U project we had to create a control panel for a ventilation system. It was important that we made it easy for the user to use, as well as quick. For the project we could choose to use either MySQL or Firestore, we chose to use MySQL as it fitted our needs better.

Tools used:

  • Angular 2+
  • Material Design
  • MySQL
  • ExpressJS

What I have learned:
Through this project I learned a lot about Angular, Material Design, and MySQL. More importantly I have learned what it means to work as a group.

Project description:
Vifttube was my final exam project for Multimediadesign. I worked with Stine (Main designer) and Sabine (Main marketing and SoMe), I worked mostly with the website and animations.

  • Adobe After Effects
  • Adobe XD
  • Adobe Illustrator

It was during this project that I further developed my skills in JavaScript and more advanced Sass.
During this project we had made a tight schedule to ensure that we made everything in a timely manner. Time management was a big thing for us in. It helped a lot, and we managed to make a project we were proud of.
